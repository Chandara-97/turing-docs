## Create a movie

```http
POST /movies
```

#### Parameters

| Name        | Type         | Required |
| ----------- | ------------ | -------- |
| title       | string       | Yes      |
| releaseDate | date         | Yes      |
| synopsis    | string       | Yes      |
| trailer     | string       | No       |
| runtime     | number       | Yes      |
| poster      | string       | Yes      |
| backdrop    | string       | Yes      |
| directorIds | number array | Yes      |
| castIds     | number array | Yes      |

#### Response

Returns the created movie object.

## Update a movie

```http
PATCH /movies/:id
```

Updates the movie by setting the values of the parameters passed. Any parameters not provided will be left unchanged.

#### Parameters

| Name        | Type         |
| ----------- | ------------ |
| title       | string       |
| releaseDate | date         |
| synopsis    | string       |
| trailer     | string       |
| runtime     | number       |
| poster      | string       |
| backdrop    | string       |
| directorIds | number array |
| castIds     | number array |

Returns the updated movie object.
