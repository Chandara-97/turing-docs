## Create Cinema

```json
**POST**: /admin/cinena/create
```

### Request

```json

		{
			"id" : 1,
			"name" : "Turing Cinema TK",
			"Location" : "National Road 6A, Kthor, Prek Leap ChroyChangvar, Phnom Penh",
			"status": 1
		}
```

### Response

```json

		{
			"id" : 1,
			"name" : "Turing Cinema TK",
			"Location" : "National Road 6A, Kthor, Prek Leap ChroyChangvar, Phnom Penh",
			"status": 1
		}
```
