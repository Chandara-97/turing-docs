## Admin/Employees

### Sign In

`POST`

```json
**Post** : user/admin/signin
```

### Request

```json
{
	"email" : "XXXX@Admin.com",
	"password" : "XXXXXXXXXX"
}
```

### Response

```json
{
	"token" : "XXXXXXXXX",
	"email" : "XXXX@Admin.com",
	"password" : "XXXXXXXXXX"
}
```
